What is AppLife Update?

AppLife Update is the most flexible application updating solution available for Windows applications.  With AppLife Update you can maintain your deployed applications effortlessly, and without administrative support.  The AppLife Update API, AppLife Server and AppLife Manager provides the tooling to create the perfect maintenance solution for your applications.

With AppLife Update, you enable your users to obtain updates to your software without asking to first determine which file to download, or whether to uninstall first, or whether to shutdown… These kinds of decisions are often the barriers that prevent users from obtaining updates.  All of this is done with a flexible solution that allows you, the software publisher, to control the process.  You can:

•      Use the AppLife Manager to update your software automatically and without any integration code within your software source code.

•      Integrate automatic updating features directly into your application using the AppLife Update API and built-in visual controls.

•      Create a completely unique, custom user experience for your application maintenance process.

•      Incorporate license or eligibility checks into the update discovery process, controlling the distribution of updates.

•      Analyze detailed information on the state of your deployed software.

•      Control access to individual update packages and individual users.

•      Coordinate between separate running instances of your application during an updating process.

•      Elevate privileges during the update process

•      Automatically detect proxy settings

•      Add custom updating activities easily using your preferred.Net language

•      Easily update applications running on Windows 10 / Windows 8 / Windows Vista / Windows 7.

 

Remove the barriers that prevent your users from updating their software.

 

The AppLife Update solution is available through two licensing options:

•      Hosted Subscription model, where you have access to a cloud-based update publish location and all the AppLife tooling for a low monthly subscription rate.

•      Through perpetual, on-premises software licensing.
