Using AppLife Manager

AppLife Manager is a Windows Service and Interactive System Tray interface that supports maintaining Windows applications with absolutely no AppLife Update integration required into the application source code.

AppLife Manager ships with and requires AppLife Server.

Software publishers can deploy applications and updates through AppLife Manager that can be completely automatic or can be manual initiated by an end-user through the AppLife Manager UI.

Once installed, AppLife Manager can be configured to check one or more AppLife Servers for Applications to maintain.
