Comparing two Update Locations

Two update locations can be compared.  Comparing update locations lets you see at a glance any missing or different updates between them. To compare two locations, click to Manage Published Updates… then check to compare locations, and select the location that you wish to compare.

 

The location selected in the list is Location A, and the location in the combo box is Location B.

 

 

The comparison dialog shows the updates at the two selected locations, along with adornments indicating information about the updates.  In the example above, you can see that location B does not contain an update for version 3.0.0.0.  In addition, there is a difference in version 2.0.0.0.  Updates can be copied from one location to another using the Copy left and Copy right buttons.  In addition, when differences are present in a specific update version, you can see the differences by selecting the update and clicking on the Show differences link.

 

 

Copying update 3.0.0.0 right, and update 2.0.0.0 left results in update 3.0.0.0 being published to location B, and the test status being cleared for update 2.0.0.0 in location A. Publishing these changes results in the two update locations becoming identical.
