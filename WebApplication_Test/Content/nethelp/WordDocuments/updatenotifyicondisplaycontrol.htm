UpdateNotifyIconDisplay Control

The UpdateNotifyIconDisplay provides an updating experience through the windows system tray.  No visual elements are present on the hosting form.  This control can be used independently or simultaneously with other update display controls.

Checking for updates. (Animated image)

 

Downloading an update. (Animated with progress)

 

Ready to apply an update (Customizable text and icon).

 

 

By default, no image is displayed in the system tray when no updates are available.  This can be enabled and customized through the control properties.
