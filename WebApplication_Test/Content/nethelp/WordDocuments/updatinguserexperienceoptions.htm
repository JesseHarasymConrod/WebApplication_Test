Updating User Experience Options

The AppLife Update solution provides a choice of options for integrating an updating user experience into your applications.  Using the visual controls included with AppLife Update, you can very quickly integrate a complete updating user experience into your applications.  Many customizations are available through the properties of the controls.

 

You can also build your own updating user experience by utilizing the AppLife Update API.  See the Custom Forms Quick Start for more information on building a customized user experience.
