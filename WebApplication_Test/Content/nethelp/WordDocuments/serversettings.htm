Server Settings

AppLife Server settings are modified through the Config view, which is accessed by administrators from the upper right dashboard menu.

Public URL – Set to the URL that is used to access the instance of AppLife Server

License Key – Enter your server license key

Require SSL – Check to require SSL when accessing the dashboard

Require Controller SSL – Check to require the clients to use SSL when accessing this AppLife Server

Help Phone Number – Set to the number an AppLife Server user would use to request help.

Help Email Address – Set to the email address an AppLife Server user would use to request help.

Smtp Server URL – The SMTP server this AppLife Server will use to send email.

Smtp From Address – The SMTP email address used to send email.

Smtp User – The user credentials used to logon to the designated SMTP server

Smtp Password – The user credentials used to logon to the designated SMTP server.

 

Setting the SQL database connection string

The SQL connection string is set in the application web.config file.
