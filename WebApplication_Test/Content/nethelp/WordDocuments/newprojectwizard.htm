Where will your updates be hosted?

Path your application will use to find updates:

Enter the network location that that your application can download updates from.  This network folder can be an http, ftp, UNC, or file path location.  It must be dedicated to this application.

Path you will use to publish updates:

This location will be used to publish updates.  You can use a different protocol to publish updates than you use to download. It is common to use http for downloading updates, and use ftp or a file path for publishing updates.

 

Verifying Paths

You can use the Verify utility to create the update location if it doesn’t already exist, and ensure that the download path and publish path are directed to the same directory.
