Manual Activation

If your system cannot be connected to the internet for activation, you may activate your license manually by using a computer that is connected to the internet or by calling Kinetic Jump tech support during business hours.

To activate manually, select the Tools..Activate menu option.  Attempt a network activation.  When this attempt is unsuccessful, a manual activation dialog is presented.  Note your Machine Name and software version number.  On a system that can access the internet, navigate to the manual activation URL (http://www.kineticjump.com/productactivation/activate.aspx).  Enter your license number, Machine name, and version number into the web form.  Save the file downloaded from the activation server and transfer it to the client being activated.  Click on the Activation file… button and browse to the transferred activation file.

You will only need to activate your software once.
