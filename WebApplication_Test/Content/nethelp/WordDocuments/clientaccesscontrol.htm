Client Access Control

AppLife Manager uses the AppLife Server Passcode Style access management system, where each installed instance of AppLife Manager can optionally be assigned a Client Access Key.  The Client Access Key is assigned during the server configuration, and can be edited through Server Configuration.

 

 

When an AppLife Manager client initiates server communications, the configured Client Access Key is submitted and compared with the Client Access Lists enabled for server applications that have enabled AppLife Manager Integration. This comparison ignores case.  If a matching entry is found for an application, the server will allow access to that application to the client.
