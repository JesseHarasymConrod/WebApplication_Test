FinalBuilder Action

An action for VSoft’s FinalBuilder build automation product is included.

The fbap file that includes the action is located at:

{AppLife Update Install Dir}\Build Automation Utilities\FinalBuilder\AppLifeActions.fbap

 

Notes:

This AppLife Update Finalbuilder action simplifies using the AppLife Update command line utility from VSoft's Finalbuilder build tool.

 

Place the AppLifeActions.fbap file into your finalbuilder [InstallDir]\ActionDefs folder.

 

Under the .Net Tools palette you will find a Build AppLife Update Project action. The location of the AppLifeUpdateCmd.exe is set under the Actions tools menu.

 

All of the command line utility features are exposed through the action interface.

 

You can find more information about Finalbuilder here:  http://www.finalbuilder.com/
