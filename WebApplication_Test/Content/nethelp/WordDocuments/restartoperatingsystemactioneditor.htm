Restart Operating System Action Editor

Action description – Descriptive text that can be used to identify this particular action in the Actions list.

 

Allow the user to decide whether to restart – When checked, this action will prompt the user with the message specified for confirmation of restart.

 

Display this message – Specify the optional message to display to the user.

 

Abort the update if restart will not occur – If the user chose not to restart the operating system, this setting determines whether the update will proceed or be aborted.

 

See the Restart Operating System action for more information.
