Creating New Applications

Administrators can create a new application.  When creating a new application, the Application ID can be set.  Once created, the Application ID can never be changed.

When creating a new application, set the application Owner.

Upon creation of a new Application, the server will send an email to the designated owner that includes all of the relevant information for accessing and using the new Application.

 

Deleting Applications

When deleting applications be aware that all update packages are deleted with the application. This action cannot be undone.  Consider transferring all published updates using the AppLife Update Manage Published Updates feature before deleting an application.
