Command Line Utility

AppLife Update includes a console application that allows you to automate performing build and publish operations on a specified AppLife Update project file (.aup)

 

You can use this console application to integrate AppLife Update build operations into your automated build process.  You can also add script to a post-build action in Visual Studio to automate publishing an update for a given build configuration.
