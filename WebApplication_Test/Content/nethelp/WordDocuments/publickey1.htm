Public Key

Sets the public key value of the application.  This value is the public portion of the private/public key pair of the AppLife Update project file used to create updates for the application.  This value is transferred automatically when the New Project wizard is used to create the AppLife Update project for this application.

Note: If this value is changed after initial creation of the update project, none of the existing update packages previously created with a previous key pair will be valid and will need be removed from the server.  This value should only be changed in the event of a compromised private key.
