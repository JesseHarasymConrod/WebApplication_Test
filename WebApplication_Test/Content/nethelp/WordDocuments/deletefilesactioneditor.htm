Delete Files Action Editor

 

Action description – Descriptive text that can be used to identify this particular action in the Actions list.

 

Target client folder – The folder on the client to place files that contains the file(s) you are removing.

 

Subdirectories – The directory below the designated client root folder that contains the files.

 

Files to delete – The filename of the files that will be deleted.  One file per line.  The filename can use * and ? wildcard characters.  Case is not considered.

 

Expansion variables can be used to set the target client subdirectory property.

 

See Delete Files action for more information.
