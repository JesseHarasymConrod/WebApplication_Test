Install GAC Assembly Editor

Action description – Descriptive text that can be used to identify this particular action in the Actions list.

 

Target client folder – The folder on the client to place the assembly for GAC registration.

 

Subdirectories – The directory below the designated client root folder to place the assembly.

 

Assembly to Install – The location on the build system where the assembly to be placed in the client GAC resides.

 

Reference Type – The reference counting scheme to employ in maintaining the assembly in the GAC.
