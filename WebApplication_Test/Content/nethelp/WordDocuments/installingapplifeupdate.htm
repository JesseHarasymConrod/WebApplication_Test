Installing AppLife Update

1.  Run the AppLife Update Setup msi file

2.  Generally, you can accept all the default options.  You can choose to install the AppLife Update application to a different folder.

 

Note:  The administrative privileges are required to install AppLife Update.

 

Upon installation, AppLife Update will operate fully featured for a period of thirty days.  To use AppLife Update beyond thirty days, you must purchase a user license.  You can purchase a license online at http://www.kineticjump.com/buy.aspx
