Editing Metadata

When an update is published, the following information is provided:

•      Version Access Mode

o  Access Control Lists allowed

•      Test Update Mode

•      Targeted previous versions that can apply this update

•      Update Summary Text

•      Elevation Type

•      Allow Chaining setting

•      Custom Data string

 

All of these settings can be modified through the AppLife Server dashboard.  Click the View Version Icon to drill into the version details.
