Update Process Permissions

Will your updates require permissions elevation?

If the users of your software applications do not hold the permissions necessary to replace your application files during an update, you can elevate the update process permissions using either the User Account Control (UAC) or by using the AppLife Update Windows Service.

 

If you plan to use the Windows Service,  you can register your application on your development computer by using the Register your program for elevation button.  Your application must be registered in order to use the service for permissions elevation.
