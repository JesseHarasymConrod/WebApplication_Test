Requirements

Client Software Requirements:

          To execute updates, and integrate the AppLife API:

Microsoft .Net Framework 2.0 or greater

          To use the AppLife Manager:

                   Microsoft .Net Framework 4.5 or greater

Server Requirements:

          To host update packages:

Any File Server share that can be accessed through a file path or UNC path

          OR

          Any Http Web Server

          OR

          Any Ftp Server

 

          AppLife Server Requirements:

                   Microsoft IIS 7.0 or higher

                   Microsoft SQL Server

                   .Net Framework 4.5 or higher

                  

Development System:

          PC with Pentium II 500 MHz or greater processor

          256 MB of Ram Minimum, 1 GB or higher recommended

          25 MB of hard disk space for the AppLife Update application and related files

          Windows 2010, Windows 8, Windows Vista

          .Net Framework version 2.0 or higher
