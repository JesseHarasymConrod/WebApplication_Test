Set Update Version

The Set Update Version action updates the AppLife Manager configuration file with version stamp of the executing update.  The action can be configured to use the active update version or an explicit value that is defined in the configuration of the update, by using a Shared Property.

 

Version Source Options:

1.  Use active update version (default value) – This option will set the value to the update version of the currently executing update.

2.  Set explicit version value – This option will set the value to the configured value defined in the Explicit version value seting

 

Explicit version value – When the Version Source is configured to use the explicit version, the value defined here is used to set the configuration version value.  Shared Property expansion is supported within this setting.
