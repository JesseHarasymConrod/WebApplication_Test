StatusStripUpdateDisplay Control

The StatusStripUpdateDisplay control provides an update experience that resides within a Windows Forms StatusStrip.

 

Note: This control does is not located in the Visual Studio toolbox.  You can add it to an existing Status Strip using the Status Strip Control pane drop-down.
