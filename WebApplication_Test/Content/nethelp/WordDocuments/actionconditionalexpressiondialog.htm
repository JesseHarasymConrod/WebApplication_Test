The Action Conditional Dialog provides the ability to test an Update Action conditional expression by defining test values of Shared Properties within the expression.

Any Shared Property utilized in the Conditional Statement is included in the grid, allowing you the ability to set a test value.

The Test button assigns the values specified in the grid to the Shared Properties in the expression, and then evaluates the expression showing the test results in the results label.

See Action Conditionals for more information.
