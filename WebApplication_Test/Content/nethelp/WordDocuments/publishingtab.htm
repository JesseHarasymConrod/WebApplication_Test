Publishing Tab

Updates can be built and published to one or more update locations. The locations can be configured from this tab.

Each location requires a URL to publish updates to.  The location can be given a name and credentials.

Verify the Update Location

The defined update location can be verified using the Verify button.  Verification will ensure that the location exists, and prompt to create it if it doesn’t.  A test file is then published to the location, verifying the ability to publish.
