Missing Action Place Holder

 

The placeholder maintains the state information of a custom action that is missing.  Once the missing assembly is located, the action will be refreshed.

 

Action type – Displays the name of the missing action type.

 

Assembly – Displays the Assembly Name of the missing assembly.

 

To manually locate the missing assembly, click the Locate the action’s assembly… button.  You can browse to the location of the assembly.
